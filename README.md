# Headphones - ALX Frontend Project

A responsive landing page for a headphones company built with HTML, CSS, and JavaScript.

## Features
- Fully responsive design (mobile-first)
- CSS animations and transitions
- Interactive hamburger menu
- Form validation
- Pure CSS shapes (pentagons)
- Font icons integration

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6)
- Source Sans Pro font
- Holberton School icons

## Setup Instructions
1. Clone the repository
2. Open any HTML file in a browser
3. For mobile testing, use browser dev tools or resize to 480px or less
